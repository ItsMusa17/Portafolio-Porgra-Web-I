Convergencia y Gemelos Digitales

Un gemelo digital es la representación virtual de un objeto al que se le incorporan datos en tiempo real que pueden captarse mediante sensores o tecnologías de análisis de datos. Se trata de una tecnología que fusiona el Internet de las Cosas (loT), la Inteligencia Artificial (IA), el Machine Learning y el análisis de datos.
